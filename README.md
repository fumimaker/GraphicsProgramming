# 13timesの遊び方
URLを読み込むとすぐにゲームが始まります．敵に当たらずに全滅させるとクリアとなります．  
パワーアップアイテムを攻撃で破壊， もしくは自機に接触させると一定時間パワーアップ状態になります．  
パワーアップには攻撃力2倍， 攻撃速度2倍， 攻撃量２倍があります．
